# Ronny Resume (Astro)

## Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Build

```sh
npm run build
```

## GitHub Pages

The workflow in `.github/workflows/deploy.yml` deploys from `main`.
If this is a project site (not `username.github.io`), update `astro.config.mjs` to set `base` to the repo name.
